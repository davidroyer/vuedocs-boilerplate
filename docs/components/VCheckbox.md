# VCheckbox
A simple checkbox component

```html
<checkbox v-model="value"/>
```

## Props

- `mixinProp1` ***String***
    
    A prop from mixinA

- `model` ***Array***
    
    The checkbox model

- `disabled` ***Boolean***
    
    Initial checkbox state

- `enabled` ***Boolean***
    
    Initial checkbox value

- `label` ***String***
    
    The checkbox label
